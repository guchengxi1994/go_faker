# go-faker

## Inspired by [faker](https://github.com/joke2k/faker), this is a faker library in go.

## Changelog

### for details, see [version.go](version.go)

## How to use

### 1.Generate a random profile:

```go
import (
	"fmt"

	g "github.com/guchengxi1994/go_faker"
)

func main() {
	p := g.Profile{
		Locale: "zh_CN",
	}
	p.Generate()
	fmt.Printf("p.ToString(): %v\n", p.ToString())
}
```
